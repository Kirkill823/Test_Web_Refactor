### Функциональные возможности:
- Просмотр информации о предприятиях на главной странице'/'
- Добавление новых предприятий через форму на странице `/add`
- Поиск предприятий по их названию на странице `/search`

### Архитектура:
TestWeb использует фреймворк Laravel для создания веб-приложения. Я решил использовать только 1 контроллер, так как в рамках проекта такого размера мне показалось этого достаточно, он спокойно обрабатывает запросы.

### База данных:
TestWeb использует базу данных MySQL для хранения информации о предприятиях. Он имеет таблицу `info`, которая содержит информацию о каждом предприятии, такую как название организации, адрес и номер телефона.

### Веб-сервер:
TestWeb может быть развернут на любом веб-сервере, поддерживающем PHP и MySQL, таком как Apache или Nginx. Рекомендуется использовать виртуальный хост для каждого домена, чтобы пользователи могли получить доступ к приложению по определенному URL-адресу.

### Тестировка :
Для того чтобы протестировать данное приложениие можно сначала создать обьект в таблице `/add`, а затем увидеть его среди других обьектов на главной странице'/' и также найти его и выписать среди других `/search`
